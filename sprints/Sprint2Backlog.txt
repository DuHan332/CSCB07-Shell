User Stories from Product Backlog that we will address in this sprint:

a) As a user, I want to save current working directory into the stack so        
   that I can change to another directory.
b) As a user, I want to remove the directory on the top of stack so that       
   I can change to this directory.
c) As a user, I want to know the recent commands.                             
d) As a user, I want to view the contents of the specific files.                
e) As a user, I want to see what I typed.                                       
f) As a user, I want to overwrite specific file with specific string.           
g) As a user, I want to create a file with specific string.                     
h) As a user, I want to add new contents to the specific file.                  
i) As a user, I want to know the description of the commands.                        
j) As a user, I want to see the error message when I give the wrong input.   
k) As a user, I want to track the whole directory tree that I have create
l) As a user, I want to find the target directory.
