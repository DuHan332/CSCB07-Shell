User Stories from Product Backlog that we will address in this sprint:

a) As a user, I want to store the result of my command into a file.       
b) As a user, I want to delete the directory.                               
c) As a user, I want to create directories as many as I want.              
d) As a user, I want to see what is in the directories or what is in their  
   sub directories.                                                    
e) As a user, I want to move the item into another directory.       
f) As a user, I want to copy the item into another directory.           
g) As a user, I want to see the content of files as many as I want.   
h) As a user, I want to download the file from a link and save it into     
  the current directory.                                               
i) As a user, I want to know how to use a specific command in JShell   
j) As a user, I want to search the item in some specific path.         
k) As a user, I want to see what I typed.
